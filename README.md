# FrameGrid

FrameGrid 是一套面向 2D 游戏的 AI 像素动画生产流水线。

它的目标不是简单地把 AI 图片缩小或像素化，而是将角色参考图和动作描述转换为具有统一像素网格、调色板、角色比例、帧锚点与图集格式的可用游戏资产。

## 当前状态

项目处于方案设计阶段。

详细背景、已有结论和下一步计划见：

- [项目上下文](docs/项目上下文.md)

## 初步流水线

```text
角色母版
  -> 分方向、分动作生成关键帧
  -> 切分与透明背景处理
  -> 统一像素网格和调色板
  -> 脚底锚点与角色比例对齐
  -> 帧间一致性与动作质量检测
  -> 人工修正或局部重新生成
  -> 导出 Sprite Sheet 和 Godot SpriteFrames
```
